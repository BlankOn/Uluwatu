# Uluwatu

Detail tentang proyek BlankOn XI Uluwatu

## Tentang Uluwatu

Uluwatu berasal dari kata *Ulu* yang berarti ujung tanah dan *Watu* yang berarti batu. [Sumber](https://en.wikipedia.org/wiki/Uluwatu,_Bali).
